\r\n\t\t\t\t\tGame needs a lot of work, but has potential. I was hoping for a woods survival simulator, but what I found was an overly complex (but not very fleshed out) base building game with added difficulties that don't help the game in either realism or immersion. The following things really should take place for realism and to make the game actually enjoyable:
1. Early game simple shelters, like a lean-to or covered trench. These don't require anything aside from some limbs or logs, and since surviving the night without freezing to death is a concern for players, a simple shelter should be accessible.
2. Traps, both simple and complex. Any survivalist should know how to make a snare, deadfall, pitfall, etc. The more complex traps are nice, but it seems silly that I can carry a bunch of string and not use it to make a snare for rabbits.
3. Some simpler ways of starting fires, like a bow drill or hand drill. Again, I'm carrying sticks, string, planks, etc. but have to use matches or make a "firestarter" to light a fire. Why not include a reusable flint and steel? To add onto this, a firebundle is another tool which could simply be added, to be able to transport an ember for a distance, to start a fire using the previous fire's embers.
4. Water is plentiful on the map, but almost too much so. Some desert map options could be neat, if some water collection methods could be added to the game as well.

As for things currently in game, that need addressed -
1. Starvation happens really fast. Probably because food is really plentiful in the forms of animals, fish, berries, etc. so it needs to happen quickly to stay relevant, which brings me to my second point -
2. Animals and food/berries are *way* too common to be realistic whatsoever. Bears every 30-40 meters, wolves sprinkled generously all over the map, domesticated chickens(?) spawning next to the player every 45 seconds, etc. It would be fantastic if the animals had more realistic behavior, (wolves being in packs, black bears being more timid, brown bears showing more aggression) were much less frequent (a kill should yield enough food to last a long time, not just a meal or two - so less frequent kills should be fine) and were wounded/killed in a reasonable amount of shots, if shot placement were correct.
3. Having random boxes across the map as the only *real* source of construction materials like nails, while requiring these materials to progress in the game is not a great plan. These items would be better off being removed from the game altogether in my opinion.
4. Similar to my last point, plant fibers shouldn't be restricted to one specific type of plant growing around randomly. It makes no sense to be wandering in search for a shrub when I have a ton of tree limbs, and various other plants around me which could do the exact same thing in real life, especially when virtually every single item in the game requires you to harvest at least 10 of these plants.

As I said, this game has potential, but needs a ton of changes to reach that potential. Right now what you have is a pseudo base building game, with some rudimentary survival game aspects that really just add up to being frustrating at best, instead of adding to immersion. It would be really cool to see the game change into being predominantly a true survivalist game, and put the base building part on the back burner. And for that, it needs realism, it needs scarcity, and it needs an extensive rework.

For 10ish bucks, it was ok. But with the game as it currently is, I won't be playing further.\t\t\t\t\t

\r\n\t\t\t\t\tSubsistence is a super fun game. It was made to be difficult, that's for sure. Instead of being deterred by a such a difficult game, for some reason, I'm challenged by it. It has it's own enticing charm to it unlike any other survival game I've played. You can play singleplayer, or multiplayer, with an open server or a private one on the latter, which makes hopping into a Discord call and playing with friends a breeze!

Bonus points for being pretty low-spec friendly! With that being said, however, when this game is cranked to high, it does not lack in detail. I think the art style is actually very well suited to the gameplay, and any reasonable gaming rig from the last few years should be able to run this pretty well with higher settings without issue.

Overall, I'd rate Subsistence a respectable 8/10. The pricing is very fair for what the game is, and it's definitely a blast with friends!\t\t\t\t\t
