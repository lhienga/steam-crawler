\r\n\t\t\t\t\tIt's a very solid game and I love the addition of the motorcycle. Collecting is fun and it's nice it shows you the collectibles on the map so you're not missing anything. When it runs well, the combat is extremely satisfying, although there are a few bugs like detection on wallrunning and grinding on bars. I love the detail in the main menu that changes locations/scenes as you play, and the graphics on max settings look astounding. Even without RTX, it has a very nice look to the game.

From this perspective, there's not much else I can ask for in a game and it's great how it's priced a lot lower than most modern games (~£30 as opposed to the very common ~£50/£70 nowadays). I can easily say I've had a lot more fun for a game worth this much than games worth double this amount.

Notice how I mentioned "when it runs well" - I'm having trouble running this on a machine that shouldn't have trouble running it at all.

It starts at a solid 120FPS using DX12 and stays that way mostly, until at a random point all my frames tank to ~15FPS for an unknown reason (and there's no consistency to when this happens - I can walk by a part of the game and it be smooth as butter, then revisit this area with the bug and it persists with a very low framerate). Sometimes they go back up eventually but there are times when it just stays at 15FPS and I have to restart the game to fix. When I switch from DX12 to DX11, the game just stutters persistently (not as bad as 15FPS, but definitely noticeable stutter and makes the game hard to play/throws you off a lot)

So I'm recommending it, but do BEWARE that this is currently a problem that exists until it's been worked on.\t\t\t\t\t
