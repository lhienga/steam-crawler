\r\n\t\t\t\t\tThe game is a lot of fun and very cool. Aesthetics are great and the core gameplay is extremely engaging.

But similar to the first, the game suffers from huge replayability issues. The cybervoid sections are better this time, basically being actual levels now, but there is still so much downtime in the game which you can't skip. The HUB area would be fine if you could choose whether you wanted access it or not. But being forced to run around and talk to multiple characters in order to get into the actual levels feels like such an easily avoidable design flaw. In general it feels like there was some kind of internal conflict with the game designers without a clear vision of what the game is what they want to deliver, with quite unusual focus on story which I think unfortunately falls sort of short.

I still recommend the game, it's fantastic and well worth the price tag. Just a few frustrating elements which I feel could be easily fixed which make the game so much better.\t\t\t\t\t
