\r\n\t\t\t\t\tI played a lot of the game when it was the online version; back during the 'live service' phase of the game's life. Back then I couldn't recommend it because the monetization was both horrific, and ruined the gameplay.
BUT for Offline they've removed all the monetization and the game plays more like an actual video game now, if you can believe that. Better still, I'm actually having more fun with the game now, and the fact that I'm not spending real world money on the game means I'm more encouraged to experiment with it. Find new ways to have fun.
It still has problems, much of the game is levelgated for no real reason, the characters being able to use any weapon makes the ability to play as who you want somewhat irrelevant, and there are WAY too many alternate versions of characters when the devs should have just made those skins that unlocked abilities, or generally make the characters more customizable. But these are, funnily enough, all functions of the old monetization scheme.
More importantly I support this because it's a great example of them having done right by the customers. NCSoft or Activision would have just cut and run, but Capcom actually gave us a better version of the game? No joke, without the gotcha mechanics and loot boxes the game is fun now, AND we can still play it. If you don't support THIS, then the next game that shuts down and you can never play again because the publisher didn't even try is on YOU. THIS is how you end a live service (although I'd still argue you shouldn't START a live service).\t\t\t\t\t

This is a bizarre one... I never played X Dive before now but I'd heard about it, and at first I was caught a little off guard by the gameplay. You really have to remember that this was a mobile gacha game and even though they've stripped all the monetisation from the game it still plays like a mobile gacha game fundamentally.

Ultimately this is a somewhat watered down and simplified X game that is best played by constantly swapping between the mouse for menu navigation and a controller for actual gameplay with a buttload of grinding in between. I'd also recommend going through the settings in the top right of the main screen as soon as possible (my graphics were auto set to low) and also scrolling through the in game settings menu as soon as possible (press select while in a stage). The main thing you should turn on is precise jumps, but I'd recommend checking the other settings too and seeing what you like.

Without a doubt the main gimmick and reason you'd want to play this are the insane amount of characters. This game is an absolutely massive love letter to not just the X series, but the entire Mega Man franchise as a whole - if you have any kind of connection to the franchise then I would recommend at least considering checking the game out.

The story is... kinda bad, though. It spends 2/3 of itself not going anywhere, finally starts developing but doesn't really make sense, and to top it all off is poorly translated and full of spelling errors. That's a recurring problem in general, really... a lot of text in the game has spelling errors or is translated weirdly. You can look past it pretty easily and figure out what it's trying to say but sometimes it can be hard to figure out what something means and it generally makes the game look unpolished, which isn't really something I expected and reflects kinda poorly on the game overall.

Despite all of this I've definitely had fun and will continue chipping away at the game to finish off the last few bits I care about, primarily Orbital Elevator mode and the various events (which you can play at any time, a very appreciated feature).\t\t\t\t\t
