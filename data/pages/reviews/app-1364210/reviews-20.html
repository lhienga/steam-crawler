\r\n\t\t\t\t\tI wanted to like this game but i'm struggling, I find there is too much micromanagement which takes the enjoyment out of planning your network and running your bus company. I appreciate the point of the game is to allow the player to control as much as possible, but I feel there should be options to automate certain processes such as allocating shifts, ordering spare parts, assigning cleaners and mechanics etc (maybe this is feature that will be added, or a feature I missed?). I also don't like having to construct the bus depot itself, maybe if it was improved then it would't be as annoying but for me i'd rather not have to expand the office size and buy a new locker every time I want to hire a new employee.

The UI could also be improved, it's not always obvious what you are supposed to look at or select to perform certain tasks, and I feel there could be better visuals to describe how your bus routes are operating. Eg it's not clear how many bus drivers you need to cover each route to run a certain frequency, maybe a more graphical timetable showing covered and uncovered duties would be better? I know i'd certainly prefer to be able to hire staff with specified start/finish times and schedule/automate which shifts/lines they will be working so you know exactly what kind of service you can operate. In the real world works workers generally have a set start and finish time with allocated breaks, this doesn't seem to be a factor in this game as all the staff appear to live in the office when they are idle, (again maybe this is something already being worked on?) I don't know if i've explained that very well but I think there are probably better more helpful ways of allowing the player to manage the network than are currently available.

Some people will enjoy this game I'm sure but I don't think it's for me. I was hoping for something a little bit nearer to how cities skylines manages public transport. Don't get me wrong I certainly enjoy some micromanagement and customisation when it's done effectively and intuitively but it's just a bit too much for me in this game.\t\t\t\t\t

\r\n\t\t\t\t\tThis game is sort of a dream I had for a long time, come true. I always wanted a game in which I could recreate a public transport system I know in real life. City Bus Manager provides that!

While there are some inaccuracies and anomalies about where it places real-life bus stops, the ability to place and name bus stops by yourself exists. That allows you to make all 20 bus platforms by the train station, if you care about that level of micro-management.
If you live in the middle of nowhere and want to make your ultra rural public transport, that's also possible. This game automatically gives you subsidies in accordance to the area (more rural = more subsidies). This is great! I like this in particular, a lot.

You can also import your own bus logos and all - although only in monochrome. But you can apply a decal on top of a decal if you want two colors, so that works. As a result, not only you can make a very realistic network, but also very realistic buses!

Now, if only I had a bit more RAM, eh? You'll definitely want more than 8GB of RAM, although I suppose this might be optimized later. That is really the only issue I've had with the game - OK, one of two issues. The other is I enjoyed it too much, and now it's late and I haven't eaten.

A dream! I'm repeating it, this game is a dream!\t\t\t\t\t

Since my last review the game HAS improved, the developers have pumped out many updates to add features and fix bugs. City Bus Manager since the release is in my top 3 games.
Introduction over here's the review:

-** Performance**
Even though I'm below the minimum requirements I'm able to have a successful experience.
Since my last review Frame Drops have practically gone, with a few happening rarely.
-** Content**
Since my last review there's been many updates, adding new buses, features and much more.
There's so much to do, with the whole world being available there's endless opportunities .
-** Realism**
Realism was obviously a priority, even if its not as realistic as possible it does pretty well
**Cons:**
-** Price!!**
I bought this on sale, at the time I thought it was the normal price, but ¬£25 is not worth for this game at the moment, sorry.
-** Passenger Frequency**
Even though we were told the game was being based of real life data i feel a stop shouldn't have 60 people on it, when in real life it has no more than 20 :smile: *i replicated irl services on the game to test this out*
-** Performance**
Even though performance is good, when i get huge there's frame drops, i feel that adding some more "Graphics" features would help

**Stuff that should be added**
- MOT: MOT once every month or 2 for a bus
- AI Competition: In Real Life your never gonna start a company where there is no competition, so add some randomly generated AI Companies. This can be an option before starting a save
- Random Events: Allow road works to be scheduled and you get an e-mail 1 week before they start. you create the diversion which stays like that for that route until road works end

**Overall**
Great game love playing it, play it on and off cause i get bored sometimes, but always come back, hopefully new game changing features will be added to improve play ability
