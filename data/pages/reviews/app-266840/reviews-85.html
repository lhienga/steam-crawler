\r\n\t\t\t\t\tGood edition of a great game, in my opinion. There's not much else I could ask for that this game doesn't already have, except perhaps some option to change the main menu UI and background at will, and have the original CD version's vanilla main menu as an option, as well as the one from the Titans expansion. But for the time being, these are available as mods.

I started playing this again in December, and noticed that the messages that appear on-screen are generally very hard to read unless on a dark part of the map, but that's about the only thing I really don't like about this edition of AoM. There ARE UI improvements in this version, but for some reason they couldn't make the text more readable by default.
Oh, and it annoys me how the lower part of the interface wasn't adapted for widescreen, it looks weird af, and personally, it's hard to get used to; they changed that accordingly in AoE2HD so I expected the same here. And they DID modify the original UI, most notably the edges, so I don't see why they couldn't make it wider.

I don't own the DLC btw, so I can't comment on that at the moment. I would buy it, not just to try out the new Chinese civ and see if it's as bad as people say, but also because it's necessary to play that amazing Aztec mod.\t\t\t\t\t
