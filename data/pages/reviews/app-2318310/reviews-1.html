Re-Up is a continuation to the original Class of '09 game- this is an entirely different game and basically serves as a sequel. Pointing that out since I didn't know if this was an extension or what it was. But it takes place in the same setting/time period as the first Class of '09 just with totally different routes/scenarios. Compared to the first Class of '09 game, this focuses more on giving longer fleshed out stories with less routes overall. (The first game had 15 endings, this game has a much smaller 7). I think this was a great decision- routes feel much more meaty and full with a lot more focus. They can get weirder and wilder and spend a lot more time with the characters (There's a lot more character variety as well in this game which is great!). There's also a lot of mini stories you can fall into along the way, as well. I really feel like this took everything good about the first game and trimmed away some of the less interesting parts. If you enjoyed the first game you'll enjoy this just as much, probably even more.\t\t\t\t\t
