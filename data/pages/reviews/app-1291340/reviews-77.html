General Information of The Review

I do recommend playing this game and I feel like I got my money worth for how inexpensive this game is, however, I do wish there was even more to this game that would get updates in the future.

I haven't played a lot of this game, since I played it during troubles with my internet, but I do intend to come back to his game and chill out working on creativity practice. I still got a lot on what I want to do, even getting into modifications to the game as to make and have different textures for different looks/themes. For example, having a more winter season to the builds I make. This game is mainly just a tech demo proof of concept type work, which is pretty well implemented rather than explicitly saying "this is a tech demo" with only basic textures that has no character to it.

Also, I know this game is described as a toy. But I call it a game due to what I consider, and what a lot others consider, a game is. Some people had complained to me about my choice of words when describing this game, but they don't seem to understand I consider a game as anything digital displayed on a monitor display where the user is interacting with the medium. Legit, some of you need to calm down because I decide to use a word to describe this product.

Criticisms of The Game

Before I get into this section, I'm happy with the game and I want to see the best for what this game has to offer. This is why I have criticisms of this game, so that maybe updates can come out in the future for improving the game.

What I'd Generally Like to See

What I would like to see out of this game is more updates to it. There is a lot of support for this game and I would love to see the base game get more features for it to be better fleshed out. Like I said before, this is basically a tech demo and it will still have a lot of work I personally view as unfinished for it to be considered as a fully finished game.

Price Not Equal to What You Get

The asking price of this game is $6, which I do think is a bit too high considering this is literally just a tech demo. I would say the better price should be $3 - $4 for what you are getting in this game. I know some may say "why would it matter if it is a couple dollars less, it's not all that expensive at $6 anyway", to which I say, you are wrong. I care about the value of the game, this being the quality and quantity of content in the game relative to the asking price. Despite the content being great quality, there truthfully isn't much for this game to ask $6 out of me. Which thankfully I got this on the Steam Summer sale for a cheaper price more close to what I would pay for this game.

If there were more features to this game, of the same great quality, I can see myself paying $6 - $10 if you added some more quality content. In fact, if you fully developed this game to be larger than it is now, I can see myself paying up to $15 for this game. However, the tech demo that it is now just doesn't seem worth the asking price of $6.

Updates I'd Like to See

Let me start off saying this, if they want to live up to the $6 asking price, they should have better modification integration for this game. This game doesn't seem to get any updates, but this one is a must if you want to meet up to the asking price and then just do nothing after that. I really don't want to have to deal with 3rd parties modifying the game as to make this game have all or most of the features I'd like to have.

In no particular order, here is a list of updates I'd like to seer:
  • Better modification support
  • Playing music while building (I've only had menu music)
  • Automatic day and night cycles
  • Automatic and Custom seasonal changes (including snowfall for winter and leaves falling on the ground for autumn)
  • Aquatic structures (Boats is a good example)
  • More animal types (no humans, but could be toggle-able option)
  • Better sharing of work, especially for cross-platform
  • Better texture pack support
  • Manual object manipulation after the automatic placements
  • Two modes, one for relaxed back play and the other for more specific control

This is all of what I can think of right now, but there is for sure a lot that can be done to this game. At least have better modification integration, so we could have modders make these sort of updates to download ourselves. I know you can mod this game, which is why I said "better" with certain points in the list.

Conclusion

I think this game was worth the money I had paid for, especially since I got it on the Steam Summer Sale. It is a fun game and I see myself playing it for more hours to come. I like to use this game to use during some downtime and for when I want to relax at the end of the day. However, it isn't without criticisms and I would like to see the best for the future of this game.\t\t\t\t\t
