\r\n\t\t\t\t\tOverall I rate this game an 8/10
As of writing this review I completed the main story and did the side quests that you are able to do while playing through the story first time (not sure if there are way more after having completed the game) on normal mode.

The game has a great story and the visuals are awesome as well. There are many points to like about the game. The materia system with the different weapons and their unique upgrades allow for many different play styles for each character. Combining this with each weapon having an skill that you can master to use it with other weapons is great.

Although the game is great there are a few problems with it. The main problem that I faced was the combat itself. The combat is fun, but badly executed. The dodge mechanic doesn't work well enough were you need a materia to actually make it work. Besides that the materia for aggro doesn't seem to work, as the enemies mainly target the character you have currently selected. Furthermore, the ATB system is good and fun to play with allowing for the tactical part in the action combat. However, the execution is poorly done due to the companions that are being managed by the A.I. not generating any ATB. I myself can easily fill 4 ATB cages before the A.I. has filled one. This becomes annoying because if you want to use magic or an ability on a character you have to swap to them and manually fight. Which is annoying because some characters basic attacks are horrible making the ATB filling not fun. Especially when you try to cast or use an action and spend ATB but get interrupted because the A.I. targets your character. What you have to do is for example switch from cloud to tifa so tifa is getting aggroed and afterwards you cast the ability with cloud. Although this is the case the game still works and the combat is fun and challenging at certain points.\t\t\t\t\t
