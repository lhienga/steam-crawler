\r\n\t\t\t\t\tThis game has a lot of potential.

As others have said, yes it is basically Rimworld with Dwarf Fortress Elements in a Medieval setting.

However in its currents state, it is missing many of the finer features of those games that gives them infinite replay value.

I have about 80 hours in Going Medieval so far and I feel like I've capped the current gameplay. Which is fine for now, this game is still early access so I expect more features to be released over time.

Some features I hope to see in the future:

Water. Rivers, ponds, lakes, moats, marshes, and FISH.

More detailed medical information and problems. Right now settlers just have a health bar - I *really* enjoy Rimworld's system for character health problems. I don't expect a 1 to 1 copy of that, but a similar system would be quite nice. I don't even think your settlers can get sick at all currently so a system to include things like that would increase immersion.

More systems for warfare. Unit formations would be cool. Siege engines would be cool. Being able to attack your enemies settlements would be cool.

More food options. I already mentioned fish, but having more options for growing different foods would add a lot of depth to the game. Some crops they could add are onions, garlic, peas, and oats.

A better tech tree would be nice too. The one they have at the time of this writing is rather scarce.

I love what they have in the game so far and I can't wait to see the game grow and evolve. I fully understand that it's early access and the dev team is small, but it's clear that they've put a lot of effort and passion into this project and I appreciate the direction they're headed.

Super excited to keep playing this game for years to come. If you like games like Rimworld or Dwarf Fortress you'll probably like this game too - but right now don't expect it to be fully held up to those games' standards.
Hopefully within a year or two it will be.\t\t\t\t\t
