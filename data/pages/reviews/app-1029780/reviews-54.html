This is an excellent game!
I've been looking for a good Medieval construction and simulation game for a long time. This has amazing potential.A few suggestions:
  • Warfare Items: Separated into: Armor, Blades, Blunts, Bows, Clothing, Polearms, Shields, & Siege Weapons; as most people don't know what the different names are.
  • Pop-up Description Windows: Enable a windows for each Item including in the scenario editor, especially for the Equipment, Force Perks, & Structure Kits.
  • Item Selections: When selecting items from Overview allow item to be installed or interacted directly without Force Jumping to the items location. It is a real annoyance to jump from you build location to the item and then have to waste time finding the build location again.
  • Roofing: There needs to be some 90 degree angle roof tiles to match the roof lines where structures extend at a 90 degree like a porch or side room.
  • New Walls: Chicken/Pheasants need Coops and other Animals need Barns. The current log walls are not appropriate for these structures and should use Boards & Planks.
