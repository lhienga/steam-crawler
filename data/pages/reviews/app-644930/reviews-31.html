\r\n\t\t\t\t\tIt is a real time strategy game with a pause function.
No PvP, but that does not subtract from the game's meta.
Humans are united against the enemy. So obviously we won't have PvP if we are united.
One very cool function is the difficulty setting. The apocalypse setting is so difficult, I don't think I will ever win on any level.
It is a bit Rogue-ish, in that you can not save the game while playing. You start a mission and play through win or loose. No do overs inside the mission, and there is a penalty for failure. However you can replay any mission set on any difficulty.
I only have one qualm. During the loading screen before and after missions, there is a message along the bottom of the screen. A communication from another land, sometimes in a different language. America's, Germany, and others. The problem is that the game loads so fast that I can not read it. A "start mission" button would be appreciated or a "continue" button, so that I have time to read the message.
It is a great atmospheric addition to the game. Reading other people's pleas for help as they are surrounded by billions of the un-dead.\t\t\t\t\t
