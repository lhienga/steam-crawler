\r\n\t\t\t\t\tGiven the game released in 2017, it is still quite good. However, I simply cannot recommend the "base game" to people, you need at least a few DLCs to fully enjoy the game, especially ATV. So, I'm going to recommend that DLC instead of the base game.

TLDR; buy on discount, if you like the game, then get the ATV DLC. Or, buy the bundle from the start.

More Rants:
At first impression, I dislike the extra maps to be in paid DLC, but, its reasonable, same for the cosmetics and dog companion. Then, I realized there's FREE DLCs, so I'm very impressed that the game is evolving through time based on real world.

Also, I do enjoy the game with realism on the hunting experience, learning to be patient before I pull the trigger and so on. Then I realized, hunting is a very expensive hobby, as you need to spend money to buy better gears, and player need to buy DLC for weapon packs to get better gears, just like in real world, LOL.

Joke aside, its a very nasty DLC, because the ATV is the only vehicle in game and this has been complained since 2017 by many players from day one. I know its not going to change soon, so I'm going to suggest this to the devs: please add one more FREE DLC to unlock a lower tier vehicle and balance it so that player need to reach certain in-game level to buy it. (or make it an exclusive event, idk)

About non-dlc weapon, the default rifle and free ammo is good to go. Most rifles are decent, yet, the meta is quite obvious, so player eventually go for the same choices. However, the handguns, shotguns and bows are mediocre.

Handguns: I was given a free Sundberg 454 and I wanted its "Tru-Vision Tritium Sights" but need a very high handgun score (too grindy), which I think is badly balanced.

Shotguns: very limited, and suffers from the same needed score issue.

Bows: needed a perk called "Recycle" to retrieve fired arrow, which I think is another bad balance to put it too far late at 3rd tier, should be 1st or 2nd tier on the perk's skill tree.

Also, the skill and perk points are gruesomely slow to grind, PLUS, the reset cost is TOO MUCH, if a beginner is testing out skill or perk, such penalty is too much, should use a cooldown timer instead of in-game money.

Also, the "Hardened" skill should also slowly regenerate health over time.

Finally, some tech issues for the devs:
- Borderless Window should be in-game option, player should not need to configure "/borderless" in launch option.
- should have a button to toggle on/off HUD icons, like hide animal clues.
- when auto-walk is on, interact should stop auto-walk (like harvest, access cache, etc)\t\t\t\t\t
