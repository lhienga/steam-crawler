\r\n\t\t\t\t\tI'll make this as quick of a read as I possibly can (TLDR Below).

The base game is fine.

This might make you think "hey, wait a second, if the game is fine then why are you not recommending it?". This is for a few different reasons. The short list it boils down to is this:

- Graphics
- Content
- Bugs

Graphics:

I understand that this game is built to support a very large number of objects on the screen, so it may not look amazing at close distances. However, it's hard to ignore how dated it looks despite how recently it was made. The base units, and their animations feel very robotic and they sometimes look like they are sliding on ice, which ruins the immersion for me. I would say the graphics are on par with games that came out around 2009 - 2012.

Content:

The devs have left the game in a pretty bad state in terms of content. The last update was the 1.0 release back in July 2023, and to my knowledge, nothing has been communicated to the community at large about another update and it has left many to ask others if the game is dead. Assuming the game is still being developed, they have not put out any meaningful features in a long time. The base roster of units included in vanilla are very stale and boring after a few hours of play. Naturally, one might decide to mod their game, which opens the door to the third section of this review.

Bugs:

Overall, the game at it's core (as far as I have played) does not have glaring bugs the cause the game to be unplayable. But as I have stated before, the game lacks in content so I decided to mod it. This is what has caused me much pain and many hours of debugging my own game just to finally figure out that this bug is something that I am unable to fix. If you decide to download a mod for a new unit with custom animations (My mod of choice in this case was a bunch of Warhammer 40k unit mods) the game only allows for ONE type of unit on ONE team on the battlefield to have custom animations. I will not elaborate a lot on this particular bug, but long story short it renders trying to play with modded units extremely frustrating and very user-unfriendly. This is only one bug, if I was able to play with the modded units I downloaded, I would probably find more bugs.


These are just the main reasons why I would not recommend purchasing this game currently. I would either wait for the game to go on sale for at least 80% off, or wait for the devs to update the game consistently. In it's current state, expect to get about 10-15 hours of playtime.


TLDR: The game is boring and buggy, wait for a sale or further updates.\t\t\t\t\t
