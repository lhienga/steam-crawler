\r\n\t\t\t\t\tThis game has come a long way since I purchased it in March, 24, 2018 and it is still showing growth. It is a very interesting and different top down strategy game and it is hands down one of my absolute favorites.... despite it being in perpetual early access (I purchased it around 0.121 and it is now at 0.3111).

The game looks amazing and is well optimized to be able to run one less powerful systems, It has a reasonable learning curve (easy to learn but hard to master) with several difficulty options as to not wall players who don't want raw challenges, and it feels rewarding overcoming tough enemies.

If you pick this game up I'd definitely recommend reading the tips that pop up on the side, they have some helpful bits of info about the game, and if you hover over the icons they'll explain what they do too (a very helpful touch).

It's a shame we will get Kenshi 2 before we get 1.0 of EotU, but I'm assuming this game is made in the developers free time for fun.\t\t\t\t\t
