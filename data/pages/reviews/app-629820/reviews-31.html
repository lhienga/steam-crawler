An insane premise makes for a fun game, but mired in technical issues

I own the XB1 version of this game, picked the PC port up in a recent sale as I thought it would be fun to play at 60 Sharks a Second.

And yes, the game is fun. It is repetitive, it is a bit grindy, the targeting could be better, but it is just good silly violent fun. Never takes itself too seriously, some funny dialogue, a decent gameplay loop that makes you feel like a beast. And you cannot deny jumping about the beaches eating humans left and right is a great laugh.


But.

This version of the game is riddled with problems.


Many 12th Gen CPU owners simply cannot start the game. There has been no official patch for owners of today's current tech and the workarounds vary greatly in success. I was lucky enough to not have this problem but instead have another.

I can only play for about an hour at a time before the game crashes with an Unreal Engine error. My guess is that it is a memory leak.

So far nothing else has happened, but other users have reported game save wipes and even total PC shutdowns from the game crashing. So not good really, is it?


I understand games, once done, are done. We're not asking for content, we're just wanting to be able to play a game on current hardware technology without issues. The game is still on sale, any stability problems should be addressed or the game should be taken off the store front.

This isn't some retro Win 95 classic, nor are we running the game on outdated hardware, customers are moving with the times yet the game is not.

However the Devs/Publisher seemingly do not give a damn. No mods active on these forums, nothing about hotfixes or similar. Zilch. Yet, the game is still on sale.


A super fun and bizarre "rage em up" akin to early GTA or Destroy All Humans, but a technical experience that really isn't worth the effort.


Only buy if these issues do not deter you, and really consider refunding within that 2 hour period.

Win 11 i7 12700k 3.6ghz, RTX 3070 8GB, 32GB DDR4 Ram\t\t\t\t\t
