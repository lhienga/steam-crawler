\r\n\t\t\t\t\tThis game is quite interesting as there are so few out there like it. The concepts of making basic logic gates and building up more complex components to eventually craft a CPU architecture as well as to be able to design or implement your own Instruction Set Architecture along with your own custom defined Assembly Language is quite impressive. The game itself does have a few quirks and some nuisances at certain parts of the game within the campaign however the ability to have a sandbox and to build your own is very enjoyable. This game isn't for everyone, yet for anyone who's intersted in cpu architectures, hardware engineering, software engineering, this game has a realatively decent overall experience. Some of the game's GUI / UI can be a bit clunky at times which could use improvements yet considering the complexity of what it would take to build such a simulator - emulator sandbox such as this is quite impressive to say the least. Hopefully with time this game will only aim to be improved. Out of 5 stars I'd give it about 4.\t\t\t\t\t
