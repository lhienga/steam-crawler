\r\n\t\t\t\t\tTuring Complete is a well-made simulator that guides you through creating a simple computer architecture, from logic gates to assembly programming. You reuse the components you made in previous steps to build more complex components. Generalisations of the components you build are often provided to save you some effort.

As a software engineer, the game echoes some classes I had a few years ago where we also built a computer with simulation software. But this game takes a much more recreational approach and provides much-needed guidance. At no point was I completely stuck, and I achieved 100% completion in less than 20 hours. You can start the game without much prior knowledge (just a solid sense of logic) but you will need some programming skills to get past the assembly parts (understanding variables, loops, and functions is required).

For advanced users, this is also a great sandbox to play with circuits, build architectures or attempt to tackle the special challenges associated with Steam achievements. As an algorithms enthusiast, I appreciated the last part which offered some algorithm challenges (simple algorithms, but implementing them in assembly is the real challenge).

As the game is still in Early Access, here is some specific feedback for the developers:
- There is often a bug with the direction of arrows shown on wires when the wires are edited. The arrow flows in the correct direction but the arrow itself is pointing against the flow.
- In the maze level, the coins count isn't reset when the game is reset.
- In multiple levels, the label "Test x/y" display under the screen in the program view breaks when testing the program
- When adding breakpoints, the game pauses not only on the breakpoint but often on the next instruction or several instructions later. As if there were invisible breakpoints.
- When switching to 4B instructions, there was no guidance on how to write assembly for this new instruction width I fund out with trial and error that one can just write 1B tokens separated with spaces or line breaks.\t\t\t\t\t
