\r\n\t\t\t\t\tSo far, played up to "Turing complete" level. The game is a really nice introduction to logic and computer architecture. The 'campaign' builds up from basic logic gates at a nice gradual pace, introducing a new concept in each level and giving the player time to get competent at using the UI and designing circuits. Completing each level is quite satisfying and the interactive nature of the the game makes building and debugging a fun process. And then there's the sandbox, where you can really do a lot of cool stuff. There are some really amazing designs shared by the community.

The UI takes a little getting used to, and I think there could be some usability improvements, but it's no big deal. There's an annoying bug in Linux where you cant use ALT to move the endpoints of connectors, but you can work around by pressing WIN+ALT. The game still gets updates so hopefully that will get fixed.\t\t\t\t\t
