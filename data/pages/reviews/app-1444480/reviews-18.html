\r\n\t\t\t\t\tTuring Complete is a nice sandbox game where you start with combining basic logical components into advanced ones and then combine these in a CPU... twice as right after using your first one to solve some simple tasks, the campaign will guide you through building a more advanced one.

This game has some limitations and ways to cheese the task scores or circuit scores overall. Within the base campaign none of these are a problem - you both neither hit the limitations nor need the cheese to get achievement scores.
The game is very addictive and it will likely make you at least start developing your custom architecture CPU or implement one of the existing specs.

What are the problems I mentioned?
- If you want to have a RAM with more than 2 arbitrary simultaneously readable addresses, you have to build it on your own with memory elements, thus you will have to do a lot of copypasting. But otherwise if you are content with 1 write 2 read, you can use existing elements just fine and set them to whichever size you want.
- Switch elements allow for constructing n-argument OR's with the delay and gate scores O(1), which likely are not applicable IRL. Neither in terms of schematics, nor in terms of physical elements used in modern CPUs. But it is up to you whether you want to use these. No one stops you from prohibiting this "exploit" in your circuits and "competing" in a separate category.

Overall, all the puzzles are interesting. Besides the binary conversion ones (I think everyone hates them). The game provides good tools to manage and verify your circuits. Sometimes you wish they were more advanced, but they still are perfectly servisable in all scenarios I have had and can imagine.\t\t\t\t\t
