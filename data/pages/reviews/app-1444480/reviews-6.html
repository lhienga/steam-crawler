\r\n\t\t\t\t\tThis is a fantastic achievement. One of the best games I've ever played.

The basic idea is it's a puzzle game. You're given a puzzle on each level, and have to work out how to get to the solution.

But the puzzles are not some abstract logical conundrum. Instead, the puzzles are real-world concepts.

So the first few levels, you're figuring out what a NAND gate does. Then, figuring out how to use a NAND gate to invert an input. Or building all the other logic gates (AND, OR, etc) from NAND.

It then gets more complicated. Can you figure out how to add two single-bit inputs together? Can you figure out how to combine that single (half) adder into an 8-bit (full) adder? What about storing bits in a memory component? Or building a RAM chip?

After a while you begin to put these components together into larger concepts. Combine your arithmetic components into an arithmetic logic unit (ALU). Figure out how to decode an 8-bit machine code input to perform instructions on values encoded into the byte, and then how to use that to perform the operations inside your simulated processor.

The next step is to build a basic simulated 8-bit computer. Out of logical components you figured out yourself from NAND gates. And then learn how to pass instructions to it using custom assembly code that you map to the binary states of the input.

There's genuinely no other experience in gaming like figuring out how to solve a maze using hardware and assembly code you made yourself. Just an absolutely incredible experience.

There are more levels. After the basic computer, you begin exploring how to build a more complex computer, able to perform more operations. But I got kind of stuck here, couldn't figure out one of the levels which is required to proceed. I hope to go back to it and solve it one day, but I had to take a break as it began to stress me out.

Very strongly recommend this one if you've got any interest in computing, or understanding a bit about how computers actually do the stuff they do.

The developer should be really proud of what they've achieved here. It's amazing.\t\t\t\t\t

\r\n\t\t\t\t\tThis game doesn't teach you digital design. After the first 10 level you might think that this is a slow enough progress to learn Digital Design from this game.... but you would be quite wrong....
You will need a book, or take a course on that topic before you could effectively progress after the first 5 level. There is no learning curve, it is more like a Heaviside step around the basic-logic end level, without proper documentation, help, or guidance.
You might even think that ok, yet it could be a nice, intuitive GUI for a digital designer.... until you notice that the logical elements' form factor not quite consistent... which could cause a serious headache when you try to make a bigger design with a large amount of basic gates. (the wiring mechanism is just awful...)

You would be far better off with a cheaper digital designer (even with circuitlab), and with a proper textbook, or tutorial, regarding the digital design and computer architecture.
In its current form this game is no more than a collection of practice examples/homework for a course (and you use a designer which is not this game....)

If you are interested in this topics, you might use the real things instead ...

(The music is also quite annoying for me... although it is similar in style to the zachtronics games, those are just much better...)\t\t\t\t\t

\r\n\t\t\t\t\tThis game is an emotional roller coaster. 40% of the time I feel clever and accomplished, and 60% dumber than a rock and feel outright hopeless.

Turing Complete has managed to boost my self-esteem in so many ways. Admittedly too lazy to dive into fundamental computer science concepts before, I found myself drawn into the gamification of CPU architecture. Much like how Kerbal Space Program taught me basic orbital mechanics, this game provides a vibrant perspective on the inner workings of a computer.

The guided journey through building a CPU is well-executed, and I'm currently navigating the Assembly Challenges section, proving to myself that anyone can grasp these concepts with the right attitude. Of course, I didn't get here alone, as many players have admitted. Some levels can fry your brain, especially if you're new to the mind-bending logic problems involved. As the game progresses, it introduces concepts that an uneducated gamer like me wouldn't likely know of. This is another testament to the power of games like this. I've been encouraged and eager to learn.

All that said, my deepest respect to the sole developer of this amazing product! I also have a heightened level of respect for the pioneers of computing for the massive brains it took to bring the technology we have today into our hands.\t\t\t\t\t

\r\n\t\t\t\t\tThis game is a fantastic introduction to computer architecture. I love how the levels build upon each other and you can reuse the components that you build. I wish that I could add new instructions from the "instructions" dialog rather than working out the instruction and then pulling up the "add" dialog. Also, it'd be nice to be able to click on a previous component (ALU, Conditional, etc) and have more information about it... for example, after I built the circuit in one challenge a few days went by and then I had to make Assembly code for my circuit, but I had to dig back into the old challenges to remember what each instruction did.

Overall, a really fantastic game - I find myself noodling on it even after I've stopped playing.\t\t\t\t\t
