\r\n\t\t\t\t\tOld childhood memories from this game. It ran great for the first 150 hours with no issues but after updating my install of windows 10 and my nvidia graphics driver it is completely unplayable. The game "Black Screens" every time I launch it after playing intro cinematic. I can see my mouse cursor and hear the BGM for the main menu but nothing else works, just a pitch black screen. After spending over a week trying to fix this, several clean re-installs and following various help and support guides, I've given up. So far from what I've read I'm not the only person that's had this issue. It's a huge disappointment because I was enjoying replaying the campaigns.
I recommend to anyone having issues with the black screen bug to try the HD Version of the game instead until Microsoft Gaming Studios finally fixes this well known bug.\t\t\t\t\t
