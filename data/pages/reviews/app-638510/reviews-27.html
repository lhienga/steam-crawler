\r\n\t\t\t\t\tI just bought the game today after trying the demo this morning. I'm still on my first run, but I've already seen enough that I wanted to give a review. Dot Age sounds like a bit of a niche concept; a turn-based survival village building roguelite. It sounds like a messy mash up of unrelated genres, but each are pulled off with surprising synergy and finesse and wrapped up in a slightly silly package.

Your goal is simply to not die. How you do that is largely up to you. Each run can last anywhere from a half hour if you die, to more than five, like the one I'm currently running. The game markets itself as being a brutal experience, but I honestly haven't seen it be that hard yet, though that could just be because I'm good at staying on top of my problems. I could see things spiraling out of control quickly if you fall behind.

The game is based around prophecies of DOOM. This comes with the plus of you knowing ahead of time when an event is going to happen. What exactly that event is depends on how well you've prepped for said event. There are multiple "domains" which build up over the course of the game. You have the ability to research and build buildings to counter those domains. If you build up enough, you can make sure that the coming event is a positive one, but if you don't prepare enough, then that event is likely going to be very bad and might kick your ass pretty hard. If you survive enough prophecies, you win the run.

Doing well in a run gives you memories, which you can use to unlock meta progression. These unlocks can be anything from new techs and land tiles, to villager traits and events. Since I'm still on my first run, I can't say for sure what the differences in runs are yet. I might update my review once I find out. You still get some memories even if you do badly, so even if you start to lose, your time isn't wasted.

Dot Age is a surprisingly fun experience. The demo is free and pretty robust. If you're at all interested in the idea, give it a try. You might find it to be surprisingly fun, like I did.\t\t\t\t\t

\r\n\t\t\t\t\tReally quite quite good.

Technically just amazing, and doubly so for a solo dev. From release I haven't noticed any meaningful bugs. On top of which the QoL changes since launch have been really cool.

There are a ton of resources to manage, but it never feels overwhelming due to a stellar UI. Every icon is amazingly easy to read, and none of the information ever feels lost.

The one criticism that I have is that the game has a janky pace if you win your first game on Normal. Since a significant amount of research is locked behind the "Memories" metaprogression, I found myself spending most of that game capped out on what I could research, and also finding it hard to plan my village as the game went on since I was unlocking things that I was blind to. It just didn't feel so great. BUT as soon as I won that run I simultaneously unlocked the next elder, so I think that some thought has already gone into this. But just in general the metaprogression is not my favorite thing.

Playing with different elders is very fun. Playing on higher difficulties actually feels good and challenging, even if it is literally just "bad number bigger".\t\t\t\t\t

\r\n\t\t\t\t\tThis is the first time I am doing a review of a game, I do so now simply because this game really deserves the attention! Incredibly fun, surprising, challenging, replayable without getting bored a second - it was even fun to learn how to play by doing mistakes and having to start over, just because the game is made with so much fun, creativity and love.

One can easily adjust the difficultylevel, but be careful - it may start nicely, but soon you will think "Oh no no no no no, what to do now, how can i still save my people" But then you can't, but then it works out yet, because of some kind of surprising intervention, or it gets even worse - you really never know. And all is packed into this creative loving world, built up differently for every game.

I also love that the game takes time, one plays in day to day turns, season by season, repeating until the final doomsday is won and all the domains challenge you at the same time. The game keeps you busy all the way, preparing for whats to come and adjusting to new circumstances. And if you think you finally understood how the game works, you have the option to choose another leading character with tricky new rules to discover and react to. It's mindblowing really.

Thank you Michele for doing such great work in creating a positive, challenging and fun game.

And: I do love the soundtrack as well!

A few game hours later: Nothing to add but one tiny thing: Just when you thought a game could not get any cuter... there came Santa and his elves! It's really fun to see Santa in this merciless game :D\t\t\t\t\t
