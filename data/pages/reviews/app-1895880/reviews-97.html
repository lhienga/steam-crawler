\r\n\t\t\t\t\tI shouldn't need to sell you on this game. If you're here, you probably already know how well received this game has been, and I think it's worth every bit of the hype.

I do need to take a moment, however, to laud how well-optimised this port is. It's insane. Nixxes pulled out every trick in the book here, to great effect. It uses DirectX12's DirectStorage API to speed up load times (important in a game designed to not have any), and it offers you a slew of graphical customisation. Perhaps the coolest trick here is the dynamic scaling -- the game can alter its internal resolution in real time to maintain a target framerate, and it uses FSR/DLSS to upscale the game back to native resolution. This provides a very noticeable boost to performance while creating only minimal artifacts, and allowing you to enjoy the game in full resolution when your PC can handle it.

I've been able to run it on the Steam Deck at a steady 40FPS with only minor visual degredations compared to what I would hope for, which is insane for a game designed to run on a flagship console from just a couple years ago.\t\t\t\t\t
