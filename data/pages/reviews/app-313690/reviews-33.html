\r\n\t\t\t\t\tLego Batman 3, Very weird storytelling, but the gameplay is actually not bad but also not the greatest. While some changes were nice such as having suits for tokens by pressing and holding the character swap button, They include both options for grappling hook from both the first and second game depending on your own preference. But that's really it.

The only negative part of gameplay is glitches, I've been finding glitches left and right when it comes to this game where it doesn't really make the game any better. There was only 1 fun glitch and that was the cyborg cyborg giant-magnet suit glitch, which if you switch from the giant suit to the magnet you end up launching yourself really high. Everything else just isn't as fun to end up in or look at.

Also, did i mention that some characters can get really annoying later on in the game? Like i felt like i was sitting through multiple sentences that sound identical to a teen titans go episode, and i really just wanted to skip the cutscenes themselves but then again it for some reason happened to be part of the story.

Oh yeah, they never fixed that same issue with it launching the game with a very small window while i can still see my desktop as if it was "fullscreen". Which again, I had to mess with the settings just to fix it. Please! Fix this when it comes to launching future lego batman games.

In conclusion, I do not recommend this game.
3/10\t\t\t\t\t
