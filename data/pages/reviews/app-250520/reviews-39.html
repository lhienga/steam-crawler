\r\n\t\t\t\t\tThis game is amazing, it is everything and more I wanted from a CRPG in a desolate setting. The worldbuilding is very good, the combat is supreme, the crafting excels beyond any game aside from really niches ones like Cataclysm: DDA. This game would be basically perfect aside from one structural issue I take with the game's economy. On it's surface, trading is pretty simple, but putting it simply, merchants will only buy a small number of items per category per merchant refresh (90 in game minutes iirc), meaning that if you are looting everything, selling it can be a huge chore of wandering between settlements. If you're open to third party tools, there is a cheat engine script that lets you instantly refresh merchants.

With that aside, get this game, it's wonderful! 9.5/10 <3 <3 <3\t\t\t\t\t
