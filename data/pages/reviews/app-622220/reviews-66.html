\r\n\t\t\t\t\tIf you're wondering: yes, this is a very good PC release. I've had no technical issues thus far, game looks and runs great, didn't have to do a thing to get my controller working, just an all around good day one release. Sad that "works on day one" is something to praise, but, hey.

Gameplay wise, this is 100% a wonderful improvement over the previous game. Combat's been overhauled, tons of really fun characters (all my faves are here!), and a bunch of quality of life improvements. If you liked the previous Extella, you'll REALLY like this one.

Music wise, it actually kind of surprised me how good this OST is... kinda regret not getting the Digital Deluxe Edition now, but maybe they'll release it as DLC eventually. Really good range of tracks to compliment every mood in this game.

And art, oh man. Wada's at it again. I never released how much I needed to see all my faves in their artstyle, but it really serves to make this game look even sleeker and gorgeous.

I'm done sucking this game's ♥♥♥♥ for now, though. Just figured I'd drop a really quick bit of praise so people know that this release seemed to have went off without a hitch. Remember to check in the steam discussions if you're having any technical issues!\t\t\t\t\t

\r\n\t\t\t\t\tBasically, an upgrade from the first game. And I mean, boy, it did get UPGRADED a lot!

- More characters. Still sad it has 1 Assassin. But at least we now have more Berserker.
- Astolfo is in the game
- MUCH MUCH better battle system and pace. I was blown away by the improvement to almost aspect of game play.
- Menus are much easier to follow and keeping track.
- Astolfo makes everything better.
- Story telling is much smoother and less.... bland. It is now kinda more exciting and less confusing? (well, If you follow the story so far xD)
- More stages so far. And by more stage, the first game use the same map AND same event for extra stage for each servant to increase the number of stage. This game only use same map with new unique event and different sector. So much more variety so far.
- Did I mentioned Astolfo?

As of the time I write this review, there is still controller issue though. Iะ kinda messed up the layout completely. It can be fixed by turning of Controller support in big picture mode. But depends on your controller, it might not be completely fixed (at least it become less messed up). If you are using PS4 or Xbox controller, you are probably okay though.

Al in all, aside from controller issue I mentioned above, everything is pretty much better than the first game. And it is not like the first game is bad in the first place. If you like the first game, this game is worth it even at full price.\t\t\t\t\t
