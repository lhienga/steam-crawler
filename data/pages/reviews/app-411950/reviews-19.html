On a more serious note, this thing is a beast. There's so much to learn but that's what makes it fun. I think that with the upcoming updates it can only get better from here. This part of the review is so short because how can I start with saying the good things about it. There's no other sim aircraft available to the civilian population (that I know of) that even gets close to modelling the F/A-18 as well as this does.

My single gripe right now with this mod was how difficult it was to get my hands on the manual. For anyone wondering where to find the manual, open File Explorer and search for the 'steamapps' folder on the drive that has the game installed onto it. Then go,

Common --> DCSWorld --> Mods (just Mods, NOT Coremods or DemoMods) --> aircraft --> FA-18C --> Doc.

Open the manual of your choice into a web browser and from there you can download it, without having to make any weird duplicated files. I also put this manual onto my iPad, so I didn't have to print 300 pages, which I found helpful.
