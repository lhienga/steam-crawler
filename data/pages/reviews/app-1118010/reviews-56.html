\r\n\t\t\t\t\tThis game requires 3 things.
1. Skill
2. Someone to explain how to play (the tutorial kinda sucks not gonna lie)
3. Persistence

This game is not one you want to be carried. This is a game where you want someone to level with you in the literal sense and this could be 1, 2, or 3 friends to tag alone but they should all used about the same level equipment as the lowest person so they can properly learn how to play the game. This game is one where even the best weapon and armor mean nothing if you don't know how to used them (or how to properly build armors). You can be new at this and learn everyone on your own but there are some thing you want someone to explain to you like the food system and how to get certain ingredients, how skill works and what skill works best for the weapon your using, and the guiding lands (this is much latter in the game btw but still VERY useful). This game will teach you patients and show you how rewarding it feels to overcome challenges. I've beaten this game now 2 times over with two characters now helping my friends play it and i can say for sure that you will build memorries that you will never forget. (like B52 and who can forget raging rajhan) To this day if a friend wanted to get into this game i would hope in and help from them from the begging because i love this game so much.

The only downside is waiting for them to watch a cutsceen before joining the quest.

10/10 would recommend this game to anyone who want to try something new and not as hard as darksouls\t\t\t\t\t
