\r\n\t\t\t\t\tThis is a great game! Don't let the graphics push you away from playing this game. There is a lot of versatility to how you approach this game. This definitely should have been the new standard for how RPGs are designed where the world in-game is affect on your choices and how you play. It is fascinating that this game came out in 2004 on Xbox. My only complaint is that you cannot scroll down to view item descriptions such as in-game books with a controller and let it auto-scroll for you. I'm not sure why that was not added to this re-release of the game but if really want to go and read the books I can switch to mouse and keyboard. However, I would much rather use the controller for everything and you cannot switch between them while the game is running as the UI and button prompts will not update to reflect them. My only other complaint is the default face for the character you play as.
In the game, some characters have a very uncanny look to them, especially the kids, which look like they will transform into some eldritch creature if you do not keep an eye on them, have this dead appearance to them like it is a puppetted corpse. This does not bother me and is more of an exaggeration on my part; however, I wish that they would try to make the main character look more like the cover art for the game.
You should definitely give this game a try, especially if it is on sale. This game is truly a work of passion.\t\t\t\t\t
