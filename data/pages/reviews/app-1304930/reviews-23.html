\r\n\t\t\t\t\tI was waiting for this game for such a long time and I am so disappointed. I never review games but I thought it was necessary for this game for others before they buy. You cannot play this game offline, at all. So, the scariest part for the hour and change I played was waiting for the "jumpscare" that would be the game disconnecting. Happened over 7 times and my internet if perfectly fine. Additionally, I thought this would be like the last 2 Outlast's but sadly this is just another major series that has forced multiplayer. If you are familiar with the Amnesia franchise, it reminds me of A Machine for Pigs, it has the name Amnesia, but it should't have the name Amnesia or Outlast in this scenario. This might just be me but I always felt that true horror games should be single player. The multiplayer aspect to this inherently removes a ton of the horror aspects from the first two games. They need to make some serious changes in order for myself and others like me to re-download this. Sad.

Update: The people commenting on this game saying its great obviously have never been true fans of horror games... This is a great idea for multiplayer, however, it strays far off the path of what any Outlast lover would be looking for. This feels like when Amnesia came out with A Machine for Pigs... it just should't have the title Outlast in front of it when this is a completely different type of game. No video cam or batteries, just some goggles drilled into your head from the start. If you are looking to play alone or have that Outlast 1/2 or Amnesia horror vibe this isn't for you. If you are a horror person with 3 buddies that also are, this is the game for you. Also, make sure you're using an Ethernet cord... 2 seconds off DC will make you exit the game and rejoin.
