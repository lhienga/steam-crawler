\r\n\t\t\t\t\tThis is the best game in VR once properly modded (sorry, HL:A), and properly modded it's maybe my favorite game of all time.

When I first modded the game, I used Aerowynx's guide, which took a lot of time and effort to get everything properly set up and running well. It also depended on some files that are no longer available, and is functionally obsolete. Got a new computer recently, and had no idea how to preserve everything.

So I went with the FUS Wabbajack (my first time using Wabbajack), which took a mere two hours to completely install, plus $6 for one month of a Nexus Premium account that I immediately canceled. And running the FUS ROH configuration the game is absolutely perfect. At first, I was frustrated that my saved games depended on mods I no longer had, so I had to start all over again. But the FUS ROH mods actually made it a joy rather than a chore to begin again, and I was surprised at all the quests I still hadn't experienced.

(If all this Wabbajack and FUS stuff is confusing to you, visit the SkyrimVR subreddit.)

After goofing around (I started the Brotherhood quest, but it took a turn I didn't want my character to follow) some, I headed off to Winterhold College -- deciding on a Destruction Mage build again, and remembering that becoming the Arch Mage scores you some items that will last you throughout the game and some cool digs.

Yes, you have to mod this game to make it really great. But once you do, it's REALLY great. And with the FUS Wabbajack it's now stupid-easy to get the best set of mods possible up and running.\t\t\t\t\t
