Now you can Search for and Disable Breakers in over 50 "unique" quests for over 8 hours of Searching for Breaker fun all while using a gun, grapple, parachute, and glider suit.



Unimaginative and repetitive sure the world is bigger - also mostly empty/useless all it really did was add more instances of the same quest type before you get to complete the story. It could have been decent if they put more work into it past the 1st hour of gameplay.
Beyond that it feels like copy/paste in new "areas" with the exact same enemies / scenery / objects / dialog in a slightly different layout with the same objective over and over.

The world could have been 1/3rd the size cutting out all the redundant breaker searching allowing us to progress to the primary missions quicker and it would have been more enjoyable... or you know the devs could have used the slightest bit of imagination and had more mission types.


TLDR Game in a nutshell:
Disable [Object] - Find Breakers
Steal [Object] - Find Breakers
Prison Break - Find Breakers
Raid Area - Find Breakers
Unlock [Object] - Find Breakers
Open Doors/Gates - Find Breakers
