\r\n\t\t\t\t\tDepending on who you are, this game is pretty easy to pick up, but for others it could be a bit more difficult. I have been playing this game for 4-5 days straight so far and i absolutely love it, i personally play with my arma 3 group and are very organised and efficient when it comes to operations which makes things like assaults on areas very fun. We do mainly run logi however and save the operations for when the activity is higher. The logistics is definitely not for everyone but in this game you have so many options of what you could do. Want to build a bunker? Go ahead and build one. Want to command a tank? Grab a tank and a crew. Want to run logi and supply the front? Grab a truck and get moving. You can do so many things in this game and its genuinely so fun. Doing logistics is definitely more fun with people you can talk to but either way i dont get too bored with it. If i was going to rate this game (and i am) i would definitely give it a 9/10
There are a few things that need tweaking, a couple bugs but thats to be expected in a game and they get fixed over time, its not like cyberpunk however. All that needs done is some reworks and with new content coming on most updates dont expect bugs to go away forever, but do look forward to the new content.\t\t\t\t\t
