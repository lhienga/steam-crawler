\r\n\t\t\t\t\tFirst of all, the game is good, compare to previous titles and series. Below here is some of my personal experience and positive point:
1) Real time combat system
This new combat system really refreshing, as the game is more towards in making decision quick and precise. The whole combat system is not fully rely on creating a powerful bomb and nuke every enemies out (like in the older series, Defeated -> back to atelier make a powerful bomb -> fight again -> kaboom -> done lul) instead, all you need to do is create a medium tier equipment, armor, and some bomb and healing items and you are good to fight all the way. I already beat all of the optional bosses and some of the elemental bosses before clear the game, if you know what you are doing in combat, pretty sure you can easily cheese through it

2) New synthesis system
The new synthesis system provide player a really easy way to create powerful items. One way is to make use the recipe change to further increase your items power and number of items used. Another thing to take note, the Item Rebuild system is really good, continue develop the items on where you left off so that you don't have to knock your brain off to find some way to loop the recipes in order to obtain best quality and best traits. Oh also the item cloning, seems like they brought back the item cloning system and all I can say about that is CONVENIENT. To sum up, the new synthesis system is so much more easy now compare to older atelier series.

3) New gathering system
Pretty fun, different gathering tools yield different items, and can be partially use as battle equipment, otherwise nothing much to say.

4) Story
Pretty straightforward, no more character endings, true ending, bad ending etc..... Story is short but wholesome, personally would prefer quality over quantity.

Well that's all about the good points of this new "Atelier". So now time for some downside of this game:
Personally, i think that they should have voiced all the dialogue in the game. Some of the dialogue is completely not voiced, while some is partially voiced, with some general voiced like um, ah, etc... you get what I mean right? As for control issue, I am used to playing atelier with keyboard so no issue regarding that. And so, this generally sums up the experience that i gain from this game, definitely would recommend this to new player who just stepped into atelier franchise.\t\t\t\t\t

\r\n\t\t\t\t\tPROS
>♥♥♥♥ ton of QoL changes compared to previous installations that I cannot recommend enough
>Decent story and progression
>A new alchemy system which I like though it may vary, it isn't puzzle mini games now.
>Love infinite item usage. I don't have to create same stuff over and over for stuff. They added auto-craft feature though, which is never needed.
>New combat system no longer requires mana, which I found to be a hassle before
>No collection limit via LP ♥♥♥♥♥♥♥♥. There is still container limit but you will be emptying it quite fast with the gem system anyway.
>No time constraint. Even if you go off and do something else its not 'canon' so its fine.

CONS
>Battle is easy to learn but hard to fully master. If only they paused when I select the action...AI was good enough though and they just become AP pumping machine anyway so didn't matter much.
>You do all these side quests to reopen clothing shop and in the end you get 0 costumes just why. The free DLC isn't even a good costume it just makes Ryza take her jacket off.
>A lot of negative reviews are about PC port performance. If you could run the nightmare port that is Atelier Firis, you can run this just fine. My specs are well below the recommended and I never experienced any issues.
>KB+M support is awkward as always. If you played any previous Atelier ports, they are exactly the same.
>More maps would have been nice, there all these different cool level designs in the game that you can see with the world-creater but they are not in the main story, shame.
>Too straight forward story to be honest
>Lack of end-game stuff. Though I hear they will add new stuff in the future.

Overall it was fun. Quite good for beginners too.\t\t\t\t\t
