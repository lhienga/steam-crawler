\r\n\t\t\t\t\tThis is the most relaxing game. Everything from the visuals to the soundtrack to the slice of life anime plot line makes it an incredible game to unwind with. The alchemy system is surprisingly robust and really fun to use as well, and the characters and their relationships are quite charming. As a newcomer to the Atelier franchise, I can say this is a great game to pick up with no prior knowledge. There's a bit of a learning curve with the battle system (that or I'm just really slow), but once you've got the hang of it, Atelier is also able to really scratch that RPG itch without ever bullying you into grinding too much. I really wasn't expecting to like it as much as I did!

That said, please note that this game definitely has a slow start. Initially the character animations will really make you feel like the game is lagging too. There is an adjustment period if you're used to being able to zip around faster and want to quickly progress through the story line--but I think it's well worth giving a real chance. Personally, once I got used to it, this game became one of my favorite things to do to relax. So yeah. I highly recommend this one!\t\t\t\t\t

\r\n\t\t\t\t\t-- GAMEPLAY --
The combat, is your regular turn based JRPG combat, made it work and look like, if it was not turn based. It is a really nice touch to it.
The crafting and gathering part of the game, can be really grindy. Especially if you want to have the best items. But, if you figure it out, after you have the extra tools, like the duplicator and world bottles, it can be fastened.
The crafting trees itself are good to use, the problem is, thatr you can't really search for recipies, you have to manually find them.
The inventory both gathering and home base, feels really limited, especially in the late game.
The exploration part of the game is pretty good. There are nicley designed areas, each with it's own uniqueness, which makes them fun to explore.
There are a lot of iunteresting, fun side quests in the game, which adds a lot to the lore and story, and there are some party quests, so your team members can learn new abilities. These can be annyoing some times, but they are worth doing them.
10/8

-- GRAPHICS --
Well, it is a console game from 2019. So it is what you can expect. It looks good, and the character design are really nice.
10/9


-- SOUNDS --
Sound effects of the game are really cute, which fits the aesthetics pretty well. THe music is really catchy and really matches the game.
10/9

-- STORY --
The story is a bit short, but it is really enjoyable, with a pretty great cast of characters. But, it is an anime like story, so if you are not a fan of those, it is not for you.
10/9

-- CHARACTERS --
We have a really colorful cast in the game, with well thought out characters. They are well written, in the limits of the game, and it is fun to get to know them better.
10/9

-- WORLD && WORLDBUILDING --
Main quests and side quests feed the lore to you really naturally, in a non overwhelming way. The world of the game is well defined, but it should be a given, since this is the 21st game in the series.
10/10

-- REPLAYABILITY --
This is a low point. There is a new game+ option, to do it in harder difficulties, but other than that, there is not much in replaying it. Maybe, if you want to pla ythe story again.
10/5

--TL;DR--
