\r\n\t\t\t\t\tI operate excavators and wheel loaders for a living. I really like this game, it is realistic enough for a simulator game in my opinion, but there are some things that can be improved to make the game a bit more realistic and fun to play.

Realism:

Reality and a simulator are always two different things, but I think there can be some improvements:

The controls on the excavators and wheel loaders (and probably the other machines too) in this game are way too fast and sensitive giving back no sense of force and weight at all. It also feels like you are digging in cotton, some force feedback or resistance in some form would maybe help that.

Another thing is about tracked vehicles, I find that you need to press the button for the left track and button for the right track EXACTLY at the same time, or else it will slightly turn, super annoying when laying asphalt for example.

Gameplay:

I don't like that the job stages auto completes - I didn't buy this game because I would want the game to finish the last 10-30% of digging/laying asphalt/pouring concrete for me, I want to finish it myself. I liked how it was in Construction Simulator 2015, where you got a message asking you if you want to continue or auto complete at around 80% progress.

Other than that I am very proud. I don't have so much time to play this game so I haven't progressed so far yet, but I'm sure there are loads of exciting stuff to come as I progress further in this game!\t\t\t\t\t

\r\n\t\t\t\t\tConstruction Simulator 2022 is a good and highly addictive game. Has a lot of good features like "Fast Travel" between many different discovered map locations and also the fact the if you are stuck on a part of a job you can just "Skip" that part. Without cancelling or failing the job.

However, this game has a few annoying and some game breaking bug (depending on your view point). Sometimes you are unable to complete certain job task due to unable to latch on material with cranes or bugs with spreading gravel with dumper trucks...etc. But the most game breaking bug is the encounter with "Error code 7" which when encountered you are unable to enter into vehicles and/or unable to use the "Fast Travel" feature. To fix it you need to close game back to desktop and reload your game and save.

This bug need to be fixed via patch! if they fix it, I would highly recommended this game.\t\t\t\t\t
