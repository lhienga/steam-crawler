No it is not as developed as Farming Simulator yet, but hoping that it will continue to improve. The biggest lack for me in the current version is poor controller support and key mapping, and no support for Trackir. A quick restart of the game always fixed the issues, and they do not come up too often.
So in short, for me this game is thumbs up and can no wait for more updates.\t\t\t\t\t

\r\n\t\t\t\t\tI'd give it 7,5/10. I've played CS2015 quite a lot (I've skipped mobile versions) and I had a lot of hopes for this game because CS2015 had A LOT of unused potential. And the game delivered. It's a really nice improvement on CS2015 and even though it didn't wow me with anything, it's improved in every aspect so I think it's worth the money.
Using machines on site feels a lot like CS2015 (which was already good) so in most cases using F1 menu for help is not necessary. There's a lot of new machine types which expanded the game nicely.
The Company aspect of the game is nicely improved too, with skills and milestones.
What I really like is that developers give even more options to do only as much work as the player feels like at the moment. Each contract can be done on Hard, Medium on Easy level, which modifies the amount of steps and the amount of work in each one. And if the player does not feel like doing one of the tasks, it can be skipped.
Also the machines can be rented as well as bought so we can do contracts that we don't yet have equipment for.
Driving mechanic is nothing to write home about, but compared to CS2015 which had the worst driving mechanic I've ever seen it's a huge improvement. Especially with a steering wheel. Too bad the mirrors in the in-cab view do not work though.
I also like that the developers had the right idea of supporting Saitek Heavy Machine side panel, which works really nicely. Unloading the truck with a tower crane using a joystick on a side panel and buttons on a steering wheel is really satisfying :) Having said that, I've played 52 hours on keyboard and mouse only and it was really fine too.

It's a solid replacement for CS2015 and overall a good game. Not great, but good :)\t\t\t\t\t
