There's no denying that this game has taken a lot of inspiration Nuclear Throne but with an added little more character and a charming and unique art style. This game is unforgiving so be warned but it's also has some interesting mechanics which make for some meaningful decisions when it comes to resource management. Think of like a spiritual successor to Nuclear Throne with improvements in just about every area. Now this game is in early access so I don't want to call it a Nuclear Throne killer just yet as there's still a lot about this game i am still to learn and and see so buy at your own risk as with any early access game but i will say this. From what I can see this game oozes quality aesthetically and the game play is really fun too. Nuclear Throne Voidigo has all the signs of a game that will be great and is kind of great already if I'm honest. We've all got excited about an early access game only for it to then later be abandoned in some kind of limbo state so again I advise some caution at least.\t\t\t\t\t
