\r\n\t\t\t\t\tI don't know what happened - this game was super fun when I played it awhile back, and seemed like the next Binding of Isaac - but when I played it today all the stuff that made it seem fun before are terrible. The powerups don't seem to work well together at all anymore. Picking a second ammo ability will ruin whatever you had going for the first if you are going for a ranged build.

In general, it seems like picking up a new power-up now ruins a run more often than it helps you, to the point where you need to open up a wiki and track if it will end your run. It's almost better to just run the game on skill alone and not pick up any power-ups, which is weird for a game of this style. There isn't enough replayability or progression to really warrant dying over and over again to see all the combinations tbh.\t\t\t\t\t
