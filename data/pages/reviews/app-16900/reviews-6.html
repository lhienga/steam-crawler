There is no grime, no trash build-up, or any ambient sound effects. The plane map is another map that feels incredibly empty. Multiplayer is relatively lacking but functional. There are multiple doors on multiple maps that have sync issues. There are not really that many servers online, but this seems to be more of an issue with the people playing it rather than the game itself.

The field of view can be customized, but the field of view when using night vision can not be changed. Low FOVs make me very disoriented very easily, so this pretty much prevents me from running any operation at night. This is a more of a personal issue, but still something to be aware of.

Only pickup if you have some friends to play it with.\t\t\t\t\t
