\r\n\t\t\t\t\tWhen this game was first released, I found it had slow loading screens before you actually played but when I changed to ssd drive it loaded a lot quicker. Directx 12 works fantastic for me with a Geforce rtx 4070 ti. Graphics are the best and the closest to real life than any other game I have played. It is a grind yes, but not a frustratingly slow progression. This game has improved a lot after when it was first released, and u can see the dev's still adding new content each week. Only played single player so no idea if multiplayer runs well or not. The game has a huge map which forces you to make a base in each biome because if u die u will re-spawn at the last bed u selected to re-spawn. I don't mind this because making new bases helps you level quicker. The game becomes more fun when you unlock stone materials. No more repairing base after storms. I tend to prefer the crossbow over guns because even the lowest bolt can kill a deer/zebra with 1 shot than a pistol that requires a few shots. better guns like the shotgun or rifle that does more damage but the process of making ammunition for them can be costly. therefore I just use my platinum crossbow with just plain iron bolts which suits me fine. I think it is a great game with fantastic graphics but may be let down with no end game. Dev's may add something later on but at level 53 so far, I don't see an end game or any goals but to keep levelling up. The materials for your base building gets better as u level and u can create some cool looking bases, but that seems to be your goal. Create bases, then create better ones, but to do so u need to travel and look for caves, to mine ores, to make bars, to make machines, to make
items. So to go to snow area u need warm clothes, so kill some deer get bones, fur, meat and leather. Make bone armour for forest, fur clothes for snow, no cloths for desert but not at night (can get cold) Even the temperature can kill you. Fun game at moment, will see after I unlock everything If the game is worth it. For the amount of hours of fun I had and still having, I think the game is worth it. They just need something to make you feel that you have completed the game. Maybe there is and I am still not fully levelled to see it yet. But I have not come across it yet. Game is not complete so one day they may work on this problem. I find that problem with all survival games except Factorio (Not much a survival game I know but u know what I mean) which has an end game.

History

Now however, more a 8/10.

The game is beautiful, even on a low-end PC (GTX 20xx/i5). I run a i7 11K with 3090, everything on Epic and get a smooth 80-100 fps, with HDR enabled.
On 74 hours of game play, I've had one crash, and no issues in game on any map.

Sure, there a DLC's with content, which I also agree, probably should be in the base game.
But even the base game made me play 50-60 hours.
