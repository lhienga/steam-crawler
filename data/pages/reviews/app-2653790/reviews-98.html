\r\n\t\t\t\t\tCool concept but once you figure out what you're supposed to do its incredibly frustrating. Spoilers below.

The actual "gameplay" is basically that its a spot the differences game. You run down the same hallway repeatedly and you need to determine each time whether or not the hallway has any differences. You have to get it correct 8 times in a row in order to win.

But it's incredibly, incredibly frustrating, because some of the changes are VERY subtle, causing you to lose your streak and you have to start over. Furthermore, there is a time limit that the game does not tell you about, so if you take too long in any iteration of the hallway then you have to start over.

The result of these two things is that you end up sprinting down the hallway over and over because you can't hit the time limit, hoping that you get 8 easy differences in a row or get lucky. If you don't, start over.

Do it again, stupid. \t\t\t\t\t
