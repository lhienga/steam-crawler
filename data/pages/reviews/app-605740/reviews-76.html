\r\n\t\t\t\t\tThe game is fun to play, but there are some things need fixing.

1. The extension ladder (on the water/pumper truck) mechanics are very wonky when positioning it.

2. Fire hydrants are very lacking in the map considering that they are required for the supply lines on the trucks.

3. The mechanics while using the stretcher need improvement. There are two grab handles at the foot and head, but you can only interact at the foot of it. Turning is quite cumbersome, clunky, and when walking behind it there isn't always animation with the legs, and should not be running all the time when going in a straight line. Should be more like the character run/walk where W+Shift runs where W is just walk.

4. The world map should be zoom capable, while I use a very large screen to view the game, I can't imagine how tough it must be to see from smaller screens.

5. For a game that has existed since 2018 there needs to be more documentation for the player. Very little exists in 2022. So that would greatly improve the user experience.

6. One of the medic scenes the victim was literally half way through the floor and upside down. I don't remember the location, but I do know it was a cardiac type of situation.

These are just a few things to help improve the game play. Hope it won't be Early access much longer. 4 years is a long time. Golf with Your Friends was the same way.\t\t\t\t\t
