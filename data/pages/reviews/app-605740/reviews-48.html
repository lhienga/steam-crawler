\r\n\t\t\t\t\tI want to like this game. I've always been fascinated with the likes of mods like LSPDFR for GTA V, or running intense calls in Emergency 4. This game attempts to fill a very niche group of emergency services that, at times, is fun.

The glaring performance issues very much hinder the experience, and any servers with 3 or more people are bound to run in to lag and connection instability. The content that is here feels empty, and while improvements have been made, it is still a very barebones game.

It is very much possible to have fun here, as long as you find a decent server with active players who use mics. The issue is even when you find a server like this, the game cannot handle the connections. With the release of 1.0, I do not have high hopes for the future of this game. Many of the issues that have existed for years are still present.

I have no doubt the developers are trying, but this game really needs significant improvement. I cannot recommend you pick this game up right now, even if it is on sale.\t\t\t\t\t
