\r\n\t\t\t\t\tI want to start by saying Flashing Lights is a great game. It has provided me hours of fun and has consistent updates. I'll start by listing the good things about it.
- Lots of vehicles to choose from
- Detailed police stations, fire departments, etc.
- Realism (in some aspects, not in others)
- LOTS of vehicle customization
- Mod capability
- Constant updates
- Other stuff I forgot to list

Now, for some issues/downsides I ran into (some of these are bugs)
- AI spawning issues, for example vehicles will spawn right in front of me as I am responding to a call
- Postal code system makes no sense, and I can't see postal codes
- Animation issues, for example holding the Rifle looks super wack
- Uniform type is bonded to the body you choose, which is very annoying
- You have to use arrow keys in menus

That's all I could think of right now. I realize it is in an early access state, however I feel like instead of releasing new features, the developers should focus on fixing the game first. Despite these issues, I still think the game is great, but maybe not $25 great, maybe $5-15?\t\t\t\t\t
