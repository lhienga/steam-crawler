\r\n\t\t\t\t\tFlashing Lights is a great concept overshadowed by greed and ultimately poor decision making.

At it's core this game get's a lot right, its fun to play with your friends, and its enjoyable to spend some time trying out emergency services as a video game. However, where this game fails is in execution. Even months after it's "Full release" this game is still a buggy mess. The servers have latency issues, the game frequently lags even on high end software and low settings, server's will randomly stop working correctly, and basic features like proper audio customisation sliders and microphone selection still have not been implemented in the main branch (or removed after being buggy in the latter case). In early access this was somewhat acceptable, but this game is supposed to be in it's full released state.

This was compounded when the Dev team started adding DLC's. The pickup truck was requested by the community for years, and when the team finally added it, they did so behind a paywall, releasing it immediately after the game "Fully released" seemingly to quell criticism that the dev team was releasing DLC's in early access. Now the team stated it took no resources from development to implement this truck as the Dev team did not actually model the truck, an outside user did, but that makes the situation no better in my mind. This was compounded with the visor lights DLC, which although free, should have simply been an update. Skip forward and the Dev team releases another DLC (3rd if your counting) even though no real new content has been released since "launch" (Even though they had stated new content was coming in both September and November). In fact all they have done since release is optimise the game, something which should have really been done before a full release and is no excuse for a game running so poorly considering (as another user pointed out) that this game looks like it was made for mobile devices.

TL;DR this game is far from complete, light on content, and actively releasing DLC's while much of the game is broken. I want this game to succeed but I cannot in good conscious recommend this game.\t\t\t\t\t
