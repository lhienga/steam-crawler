\r\n\t\t\t\t\tThis is a game I wanted to like but just couldn't. It has a lot of good ideas, but the implementation doesn't work.

Pros:
The aesthetic is gorgeous
The plot is interesting
Each new resource feels unique

Cons:
The biggest issue is that the gameplay loop is not fun. It's not an idle where you leave it alone, as your machines need to recharge every few minutes. So the gameplay loop ends up being moving from one machine to refill to the next for hours on end.

It sounds like eventually it becomes an idle, but personally I'm not looking for an idle as letting something sit for a day isn't fun to me. Rather, I like the games that are active "idle" games, like Forager, Magic Research, and Orb of Creation.

The second biggest issue is the noise. And I don't mean just the sound. The entire game seems to hinge on the user experience becoming worse and worse. There's the actual factory noise, but there's also the screen clutter as resources move around the screen, and then eventually there's a honk noise, flashing, and time slowing down or speeding up. All of these things get fixed by new machines, but therein lays the problem. Instead of fixing gameplay issues like gathering certain resources too slowly, most new "one time only" type machines are devoted to improving the headache inducing user experience issues, which is a terrible idea.

Along with the intentional bad user experience issues, the game suffers from some unintentional ones as well. First, many buildings are too tall and block your view of other buildings. Adding onto that, to interact with buildings you must click their bottom, which is often obscured. Second, the machine menu is designed horribly as a single giant list instead of broken out into categories. Third, there's a copy mechanism that rarely works, as if you copy a second or third generation machine it will try to copy that machine and fail because you have to put the other machine generations down first. To be fair, the creator has addressed this and said it is on his list to do, but IMO this is inexcusable in this type of game.

The next issue is that a lot of the fun of these games is in redesigning the factory, but I found that wasn't fun at all in this game. While you redesign, the rest of your factory sits idle as you have to ignore all of the resources that you aren't recharging because you are too busy. Meanwhile the sounds and colors all around you continue, distracting you from getting in a good design flow. Personally, I think the game needs some sort of build mode, maybe an FTL like pause feature, where you could pause all the UI noise and stop resource gathering to edit your setup. To add to the frustration, prices ramp up FAST to where you can only build 10-20 of most buildings, so you aren't going to be spending much time designing as there won't be much to design, so mostly you just sit around refilling buildings, which is tedious and not fun.

The final issue is that like a lot of games in this genre, the pace starts out good and then slows to a crawl. Where I stopped, it was already starting to become a few hours between story bits, which when those few hours is spent refilling buildings, is not a fun time.

Overall, this game has a really great foundation with the art. However, it massively needs a UI overhaul, the intentional UI annoyances should not be fixed by building machines. It also needs a gameplay loop overhaul. I'd say add a build mode that pauses resource gathering, allow a lot more buildings to be built, and lengthen the time that things needs to be refilled.\t\t\t\t\t

\r\n\t\t\t\t\tI dislike idle games that go on forever. I don't want to log in every day to collect what was gathered overnight, or to be pushed to buy gems to make progress go by faster. This game doesn't waste your time (despite being an idle game). It stays interesting and engaging up until the ending (and it has a meaningful ending!). Sixty Four knows what it is and it knows when to stop. The achievements are fun to go after as well (except for the achievement for playing for 64 hours... now I guess I just leave the game running for another 30 hours?).
Overall a great game, highly recommend if you enjoy clicker or automation games.\t\t\t\t\t
