\r\n\t\t\t\t\tI paid around 5 bucks for this game. It is a great game however since i played L4D2 a lot. I feel like this game is just very boring.

What they did was basically create a L4D clone and make it a bit modern with a card system to change stuff up. The idea is creative, but for a player like me, it just brings me back to L4D2. Everyone should try this game but only on sale and if you need to start with a zombie game. Try other zombie games like their previous game L4D2.

The combat exists and the zombies exist, and it is all great because it all comes together. However it does not really add anything new to the table. I tried the card system and it is great. However you can run through the levels without even thinking about it at all at Veteran Difficulty.

Overall, it is worth a try but the game just feels mediocre at best.


Also, after playing for a while, you see how low quality the game turns out to be. The same idea, style, and animations are used from L4D2 and it just feels really clunky for a game that is attempting to be moderrn. If this game is modern then please make it modern.

For example, At the start of Act 4, you chase an abomination and all the bots on my team died because they went under the monster. They were not crushed at all but they died and it happened to me as well.

Or ladders only work if you climb from the bottom when you are walking instead of clinging on to it when jumping. Or how animations that are in 3rd person still show the crosshair on top of your character which is very weird? The game uses tech and ideas which are quite outdated for a game trying to be a better version of L4D2.

I suggest you all to stick to L4D2. The game is trying to be unique but fails because it cannot disattach itself from L4D2. It also makes L4D2 extra complicated by adding a card system that only a few in the community care about. On top of this, the game is abandoned by the developers making this purchase questionable. Before purchasing this, I thought I would have a better L4D experience but now I just feel annoyed because I bought a below average game.\t\t\t\t\t
