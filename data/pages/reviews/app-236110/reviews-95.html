\r\n\t\t\t\t\tYou have to have good network to play this game.

I thought the concept of 'private games' meaning I could play a local version of it, but I was mistaken. Even for the private games it would need to connect to the central server and maintain a continuous connection.

Maybe I would think highly of the game if it was playable for me. To be honest, I think the graphics and gameplay are totally fine.
I live in a place where the ping to this game's servers are high, and this game will randomly hang, teleport your character around and do all sorts of mysterious stuff.
This game is (currently) very unforgiving for the players with high ping, and the server-client model does not handle high latency well. I have played other games with high ping, but it does not affect overall experience because the client knows how to smoothen the 'time gaps' caused by high latency well -- this game doesn't. Having your character 'blink' around because of sync issues between client and server is very annoying.
I think if a game needs a constant network to play, it should set up more servers across regions to improve the experience.\t\t\t\t\t
