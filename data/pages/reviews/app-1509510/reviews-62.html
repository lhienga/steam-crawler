\r\n\t\t\t\t\tI cannot recommend this game in spite of the fact that it looks extremely interesting. I've been playing sandbox games for years, but this one has been frustrating and very hard to play, even on "Easy Level". I particularly like the idea of a "tech tree" but the tech tree in this game is a complete disaster. It makes no sense at all.

For example: Players begin with the technology to build a house but one of the "Livability" techs is how to build a "Tent". If you know how to build a house, it doesn't seem like a tent would be terribly difficult. But, in order to make an "Advanced House" you have to waste a tech point on learning "Tent". Ludicrous! You can't have a farm without "Compost" first. You can't have a "Fishing Port" without having "Smelting", like the metal, not the fish.

The "Help" section is a joke. Admittedly, I miss the ability to buy a game guide when a game is released, but I expect to be able to look up necessary information in the help pages provided with a game. Not so here.

I'm moving Settlement Survival to "Hiddden" on my Steam interface so it cannot annoy me any more.
