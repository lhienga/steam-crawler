TLDR : Banished with Anno lite!

The game is a City/Settlement Builder with survival aspect into it, much like Banished.

If you have played Banished before, this is basically Banished but better. Why? well for start, the game give a lot of more information about the game to the player like the more information on settlers individual info compared to banished, more intuitive UI which tell more information like crop's yield per square, grow time, and temperature resistance etc. compared to Banished, time scrolling info which will tell the events in the next 20 months, some random event through your gameplay which spice up many things, and many other small QoL change/additions.

The game also have many feature which is absent in Banished like Item Processing and Logistics (which is why i called the game Anno lite), processed goods have more benefits, for example processed food doesn't have health penalty like raw food and they also fills your settlers hunger better so they need to eat less, another benefit for processed goods is that it have more value so they sell better than raw materials. Oh yeah talk about selling stuff, this game actually have currency! unlike in Banished which you can only barter with the traders, you can buy stuff with money. You get money from trade with the 4 different factions, each faction have their different benefits if you make them your friend by keep trading with them, for example the European style faction will give you their hard wood plantation and hard wood tools technology, hard wood tools are MUCH better than your standard iron tools since they have more durability = last longer and also can be sold at higher price compared to your standard tools which is useful for trading to the other faction to increase their relationship. The Arab/Persian/North African style faction have the most OP technology if you trade with them!

Another feature which i really like is the Tech Tree system which add some sense of progression, i don't really like that in Banished everything is already unlocked from the beginning but in this game you have to actually unlock things as you play, while the tech tree isn't perfect since there are some technology which you desperately NEEDS such as the herb plantation and seed which you need several unlocks before unlocking them is annoying, but overall it's pretty nice.

As much as i love this game, not all games are perfect, including this one. There are many things that still needs to be fixed, my biggest issue about this game is the lag problem in the end game (3000+ Settlers on my i7 7700HQ Laptop), also the information graph about the resources production which still not optimal according to many players, there are also some other bugs in this game but be aware that this game is still in EARLY ACCESS so it is to be expected and i hope the devs fix them (The dev team are very active about updating this game so kudos to them!)

There are still more addition which i don't really explain such as "Story"/Challenge mode since you can experience them yourself but These gameplay additions is what make this game really shine compared to Banished which i also really love.

For an early access game, this game is great!

~ 8.5/10\t\t\t\t\t

\r\n\t\t\t\t\tPlaying in June 2022, with game still in Early Access.

Warning: the tutorials alone for this game last more than the Steam 2 hour trial period. By the time you've finished the tutorials (which are basically "Build this, now built, that, now build one of those ...") you can no longer get a refund and you haven't even played the game yet.

The goods: I wanted to like this game. I really did. I like village-builders with good 3D graphics which show all my little people scurrying about doing things. The graphics are colourful and detailed. The controls are fine, with bonus marks for the key which makes trees invisible so you can see everything below the tree canopy. I had zero crashes or hangs. Games load quickly. I did feel some kind of bonding with my little people, worried when they were sick, etc..

The bads: it's too slow (the gameplay, not the graphics) and too complicated. Too many different types of buildings (fields are not farms and cannot be upgraded to them), too many different types of resource, too many different types of building, production chains are too long, research tree is too complicated. I needed more adults to do work in all the different types of production building. I had new children born ! That should help ! Three hours of gameplay later, my new children were still in school and not yet old enough to do work, and I'm still struggling along shutting down one building so I can staff another. Yes, I can put it on double-speed or faster, but what's the point of having a game which is no fun at 'normal' speed ?

Programmers, artists, etc. are obviously competent. But the game needs cutting back or slimming down or something.

However, it's still in Early Access. Perhaps the programmers are proving they can make every feature work that they're going to need. I'm going to put this back on the shelf for six months and see whether it has improved.\t\t\t\t\t
