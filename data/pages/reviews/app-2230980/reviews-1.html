\r\n\t\t\t\t\tI'm a huge fan of Sid Meier's Pirates, so I was really looking forward to playing this game. But I didn't get very far.

This game has issues with simple player controls that prevent you from navigating NPC menus and playing the game properly. There is no mouse support (Why?!? This is 2024, not 1980!!) and the arrow keys are supposed to allow you to navigate through the list of options - but in my case they did nothing. I was forced to choose the first option on the list every time (the default) by pressing return. There is also no way to exit the game when the NPC dialogue options are on screen, other than to use task manager to kill the process.

The game engine is a modified version of the engine used for the developer's older pirate themed games. It looks and feels like a bit of a mess. For example, when you run the game, the first menu has mouse support. When you click "Play Game" it launches another executable and the mouse support is gone. You talk to an NPC but can't choose any options other than the first one on the list. And then it runs yet another executable.

I tried this game on 2 different computers and had the same results on both of them. Both of my systems are running Windows 11, so maybe it's a compatibility issue? Running the game in compatibility mode didn't work either.

I finally gave up and requested a refund.\t\t\t\t\t
