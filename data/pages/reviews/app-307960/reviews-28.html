\r\n\t\t\t\t\tIf you're new to the genre like I was, don't start alone unless you're a real masochist.

It is rewarding when you start getting it. But it takes time. 270 hours running I've mostly flown bombers and attackers because the skill required in fighter combat has seemed overwhelming. Things to learn at the same time:

- gunnery
- spotting
- identifying friend or foe
- tactics
- engine management and flight characteristics of different planes
- the mission/maps you're playing, navigation- and strategy-wise

You get all this difficulty and whatever I'm forgetting, while the game doesn't really offer a way to quickly check what you might've done wrong once you're shot down - if you want to check the replay, you needed to be recording in the first place - and it could take half an hour before you get in contact in the realistic servers...

I could go on about the hard stuff, but check it out once it's on sale. Just a mouse will do, clumsily, to see if you're into it. With just the base game you can drive a tank as well if that's your kink.\t\t\t\t\t

\r\n\t\t\t\t\tI really wish I could give this game a thumbs up. It does so many things right, optimization is great, plane crashes and damage is way beyond anything dcs world or anything else has to offer. This game is competing with handful of other flight and combat simulators when it comes to my time and interest. This game is missing basic quality of life features that are now becoming a requirement for me and many people if I am to even consider investing my time into it. It's time to come out of the dark ages and add these improvements or it will be left in the dust. The #1 thing I am complaining about is the inability to save different control input profiles. I'm a flight sim enthusiast so in the real world I own 5 different flight sticks and 3 different throttles I cycle through these depending on the plane I am choosing. This is not possible without spending hours of remapping with the current setup. In fact if you are a owner of just 1 hotas setup it's STILL going to be annoying. Every plane has a different cockpit layout and thus every plane needs to be setup differently but in this game they only give you 1 profile to work with. Every time you get in a new plane I have to remap the buttons the new way I want it. Oh, and there's no input search function and if you put the same input multiple times on different ♥♥♥♥♥♥♥♥ it will not warn you of contradictions or complications. The control mapping is a thing of the ancient past compared to DCS where you can map every module to your exact liking, save the profiles for every individual throttle and stick you own. The final complaint is there are no clickable cockpits, this is almost universal in all modern flight sims, it makes life easier, especially seeing that this game has garbage control input options, it would have cut some of the annoyance out of that but nope, doesn't exist. This game has vr but I can't use my valve index controllers to click any cockpits. This company keeps pumping out new modules but like all these flight sim companies they don't seem to care about quality of life addons. So i'm done sitting here and spending hours remapping my controls every time I want to try a new plane. I don't have the time for that. This game needs to adapt to the new standard, if it had these basic features it would be my first choice but for now it's my last till if anything ever changes.\t\t\t\t\t
