\r\n\t\t\t\t\tWhile I am having fun playing this game as of the writing of this review I cannot recommend this game.

This game has been in EA for 9 years now and I have spent half my play time either trying to get into servers or getting booted out at random, the server going down and booting everyone. When you do finally get in and play the game you can't see a thing. It's weather system while pretty makes this game a miserable experience. It's either foggy so you can only see 5 feet in front of you, normal amount of rain that happens to make the game pitch black, a storm that funny enough has more light due to the lightening flashing from time to time.

Then if you're lucky to see how pretty the game is and none of the above are happening well it's night time so you can't see ♥♥♥♥ still.

This along with how utterly unstable the game is makes the games experience feel like you're setting up a massive mod list on skyrim and a mod is causing a crash so you're about to spend the day trying to get the game to work.

Hopefully this is just an issue with a latest patch causing instability because if this has been happening since day one then this game is ♥♥♥♥♥♥ for good.\t\t\t\t\t
