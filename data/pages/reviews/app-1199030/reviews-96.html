\r\n\t\t\t\t\tTainted Grail is a roguelike deck-building game set in an Arthurian fantasy wasteland backdrop. The deck building aspect is fun and not as restrictive as something like Slay the Spire or Griftlands. Most of the time you can build a coherent deck with some synergy. The unlockable upgrades are pretty impactful. Each class plays quite differently so there is a lot of replay value. The game is quite stable, I had no crashes in 20 hours.

The game clearly needs some more development time. Classes are wildly unbalanced; I struggled over and over before I could eek out a win with the Wyrd Hunter, and then cruised through a run taking almost no damage with the Sentinel. Several quests are bugged, and some features are not yet fully implemented.

Overall, buy the game and play it in a few months.\t\t\t\t\t

One of the best deck builders

Gameplay loop:

1) Start in Town. Do you typical RPG hub things that you do.
2) Explore the map
3) Encounter quests that span multiple playthroughs
4) Kill monsters
5) Fight Boss
6) Repeat step 1-5 three times
7) Kill the big big boss
8) Try a new run with one of it's 9 unique classes

Why I recommend it:

* The 9 classes in this game all play differently
* This is the only game where I like to play all the classes. There isn't one class I found boring. Each of them present a new deckbuilding challenge
* The story in this game doesn't obstruct the gameplay. And since the quests can span multiple playthroughs there is always a carrot on a stick for you to chase after.
* There are an ocean of things to unlock and the progress is perfectly tailored that you always have something to look forward to. That one more run itch is present here.
* [August 2021 update] The game is a little on the easier side once you get going and know the system. Now there are 3 more modes to challenge the player. While this doesn't increase the challenge it does allow new ways to experience the game which will increase it's shelf life.


I give this game 4 candles out of 4.
