\r\n\t\t\t\t\tYet another solid AC game. A bit different this time, but like always, you pretty much get what you pay for with games from this series. This time they had completely gotten rid of the tailing and eavesdropping stuff, and pretty much all the missions very really simple, either killing or fetching something. So it can get a bit repetitive since it is a pretty large game too, but I also liked it because I used to hate those kinds of missions in the older games.

The story is decent, but you don't really get deep enough to actually feel for the main characters and there's too many of the bad guys and you don't really get to know most of them either before doing away with them so you don't really get to build any sort of strong feelings against them either. This is the one aspect in this series that has really gone downhill basically since the Ezio trilogy (even though AC3 was still pretty good too).

\r\n\t\t\t\t\tSteam Deck Users:

UPLAY does not work for this game, so you will need to install Ubisoft Connect to run the game. Fortunately it is an easy work around. Procedure as follows:

Go to desktop Mode.

Press the Steamdeck button in the bottom left corner of the desktop to open the application launcher. Go to utilities and install/open "protontricks". If you don't have it installed, you can install it through; application launcher> System> "Discover" app. Protontricks is like game genie for proton. From protontricks you can install all the closed-source system runtime libraries that proton needs to run games correctly.

Once proton tricks is open, choose the steam install you want to modify, in this case AC: Origins, ok. In the next screen choose "install an application", ok. Next screen, scroll all the way to the bottom of the list and check "Ubisoft Connect". Hit ok to download and install it over the obsolete UPLAY launcher. Once it finishes, uncheck run "Ubisoft Connect", then cancel out of proton tricks to close it.

Now, just go to play the game in steam, and it will launch Ubisoft Connect instead of UPLAY. Login to bind the game to your account, and you are good to go!

Note this works with all the games on Steam Deck that still run UPLAY, but you have to install and update Ubisoft Connect for each game to get them to work properly!\t\t\t\t\t
