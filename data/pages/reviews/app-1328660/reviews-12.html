\r\n\t\t\t\t\tGood game but some downsides.

Graphics. Not really great for a "remastered" title. Especially anti-aliasing. TAA(default) makes the game look smoothest but at the same time too blurry. FXAA and SMAA1X just doesn't do it right. I've tried Nvidia Control Panel but didn't find it satisfying either. (I've tried both overriding with Control Panel FXAA and enhancing to x8. Both were unsatisfactory.)

Input lag. I saw some other reviews mentioning sluggish controls. Now I get what they meant. As a workaround, go to Nvidia Control Panel and set Low Latency Mode to On. Honestly I'm not sure if this is really working, but it felt better after I changed it. I mean, when a game uses Vsync, Low Latency Mode always works.\t\t\t\t\t

\r\n\t\t\t\t\tWhen the original was made in 2010; the devs were making some bold moves that ended up making this game worthy of a remaster... which was probably a safer move for them considering the other modern NFS titles.

While this game doesn't have much to offer for a "remaster", it's certainly worth a shot if you don't have the original (like me). The engine sounds here are great (same with the soundtrack), there's attention to details on the cars (the info, the damage, etc), the gameplay tends to be kinda sluggish and rubberbanding is an issue.

Crossplay can make up for it at times, and if you're willing to look past the EA launcher being a requirement, then the game can keep you entertained for a while.

In short? Get it when it's on sale.

7.5/10.\t\t\t\t\t
