Mod Support
Please bring back mod support! It made the game much more fun!
Henchmen
Please allow the ability to hire ALL henchmen in the previous game at the same time.

Please change some of the attack animations and add weapons. So boring, for example Eli no longer shoots with the gun but has a gun? What's the point! Espectro uses punches, just standard attack animations. Magnolia has no weapons but uses punches as well.
Minions
Please allow more than 300 minions! The base can be built to accommodate more than 300 minions! For my base I only use 1st and 2nd floor where all my minions are. Didn't even use the 3rd floor.

Please change the attack animations as well, so boring. All of them attack the same way, even the martial artist. In the old game, all of them had different attack styles which was super logical.

Minions are way too weak as well. Older game, when in Alert Mode even the valets put up some form of a fight.
Items
Need more items which help with the management of the base. In the old game, there were two items I can recalled which help. One is better scheduling which helps with the staff rotation for the cameras. The other item is the loudspeaker which helped with the protection of the base.

Bring back the hidden machine gun and hidden cameras! Portal Pack items are practically useless. Except the portal itself. That's super useful if we can have more.
Base
I think it's a lot of wasted space with the new concept. Should enable the indoors and outdoors as well. In some way, it enables a better control of the flow but it makes it too predictable on where the agents will be coming from. They only come from 1 direction this way.

Reduce the space of the indoors and make space for outdoors. Also, the base has a hotel but guests can't stay! Another wasted potential for revenue!
Quirks / Animation
I always look forward for interrogation because in the old game, the interrogator will dance, or smash the drum cymbals in the face of the victim. Now it doesn't have that, boring.

We could interrogate using the research devices because it was logical, now we use traps. So boring, need more methods of interrogations.

When a new minion is generated, now they spin into it but old one was like jumping out of the skin. At least make it a bit more fun.

Also please make putting out fires a bit more smarter like the old game! Please enable more minions to use the fire extinguisher! I've so many massive fires with the fire extinguisher right next to it and all the minions just passed it doing their chores. So irritating!

Please also bring back CTRL + Tag like in the old game where the player is able to quickly tag all the agents at the same time instead of one by one. I know the automatic tag is already there which is good. But the CTRL + TAG is a bonus!
Super Agents
Super Agents are fine, but the ending is nothing special. In the end they all died via a trap which I think is rather boring as well. In the original version, they were defeated by their own game. I clearly remember the Super Agent Chen which was defeated in a Martial Art challenge. Made absolute sense!
Side Stories
This concept is good, helps focus the narrative. But some of them are way too long and unnecessary. Especially when chasing loots and super agents. Please shorten some of them or perhaps add more loot!

That is all I can say for now, it has so much potential. I hope these types of items can be added so I will play the game again.

Thank you.\t\t\t\t\t
